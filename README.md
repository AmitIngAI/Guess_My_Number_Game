## 🌟 Overview

**GuessPro** is a cutting-edge, feature-rich number guessing game built as an MCA Final Year Project. It combines engaging gameplay with modern web technologies, offering multiple game modes, real-time statistics, achievements, and a stunning user interface with smooth animations.

### 🎯 Project Highlights
🎮 4 Game Modes → Classic, Timed, Extreme, Custom
🏆 15+ Achievements → Unlock badges & rewards
📊 Real-time Analytics → Track your progress with charts
🎯 Daily Challenges → Compete for daily rewards
⚡ Power-ups System → 50/50, Reveal, Extra Time, Skip
🌙 Dark/Light Themes → Comfortable gaming anytime
🌐 Multi-language → English, Hindi, Spanish, French & more
📱 100% Responsive → Play on any device
💾 Progress Saving → LocalStorage persistence
🎵 Sound Effects → Immersive audio feedback

### 🎯 Why GuessPro?

| Traditional Games | ✨ GuessPro |

| ❌ Single Mode Only | ✅ 4 Unique Game Modes |
| ❌ No Progress Tracking | ✅ Detailed Statistics & Charts |
| ❌ Plain Interface | ✅ Stunning Glassmorphism UI |

| ❌ Desktop Only | ✅ Mobile-First Responsive |

## 🎮 Game Modes

### 🟢 Classic Mode
Difficulty: Easy
Attempts: Unlimited
Timer: None
Range: 1-100
Hints: Available
Points: 1x Multiplier
*Perfect for beginners. Take your time and master the basics.*

### 🟡 Timed Challenge
Difficulty: Medium
Attempts: 10 Max
Timer: 60 Seconds
Range: 1-100
Hints: Available
Points: 3x Multiplier

*Race against the clock! Can you beat the timer?*

### 🔴 Extreme Mode
Difficulty: Hard
Attempts: 5 Only
Timer: 30 Seconds
Range: 1-500
Hints: Disabled
Points: 5x Multiplier

*Complete daily for streak rewards and exclusive badges.*
